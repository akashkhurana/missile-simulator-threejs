# Missile Simulation ThreeJS 🚀

Advanced 3D Surface-to-Air Missile (SAM) defense simulation built with **Three.js**, **GSAP**, and **Vite**. Experience high-fidelity tactical engagement sequences with a cinematic HUD and realistic intercept physics.

![Simulation Preview](https://raw.githubusercontent.com/akash/missile-simulation-threejs/main/public/preview-hero.jpg) *(Note: Replace with your actual screenshot)*

## ✨ Key Features

- 🎯 **Tactical HUD**: Real-time Heads-Up Display featuring radar monitoring, aircraft telemetry (altitude, speed, heading), and missile tracking data.
- 🌫️ **Atmospheric Environment**: Procedural military base environment with concrete pads, fencing, support vehicles, and a distant mountain range.
- ☄️ **Dynamic Ballistics**: Realistic missile pathing, interception logic, and cinematic slow-motion engagement sequences.
- 🔭 **Advanced Visuals**: Detailed particle effects for smoke trails, screen flashes on impact, and smooth camera transitions.
- ⚙️ **Real-time Controls**: Adjustable simulation speed and missile performance parameters via an integrated control panel.

## 🛠️ Tech Stack

- **Core Engine**: [Three.js](https://threejs.org/) (r170)
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
- **Frontend Tools**: [Vite](https://vitejs.dev/) for blazing fast development
- **Language**: Modern ES6+ JavaScript
- **Styling**: Vanilla CSS with Orbitron & Share Tech Mono typography

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/akash/missile-simulation-threejs.git
   cd missile-simulation-threejs
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🎮 Controls

- **Restart**: Reset the simulation state.
- **Missile Speed Slider**: Dynamically adjust the velocity of the interceptor for cinematic timing.
